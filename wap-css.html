1.当文本溢出：text-overflow

text-overflow：显示省略符号来代表被修剪的文本

text-overflow: clip | ellipsis | string ;

clip：修剪文本; ellipsis：显示省略符号来代表被修剪的文本; string：使用给定的字符串来代表被修剪的文本
